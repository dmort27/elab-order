1.  kuv txiv npe hu ua = Ibraham Ali Jama
2.  kuv txiv yuav tsum tis kuv lub npe hu ua  = Mohamed Ibraham Ali
3.  kuv yuav tsum tis kuv tus tub lub npe hu ua  = Yusuf Mohamed
Ibraham
If you look at the name in #3, the first name can be any given name,
but the middle name must to be the father's first name and the last
name must be the grand father's first name.  this is true for all and
this is how all muslim name their kids one generation after the other.
