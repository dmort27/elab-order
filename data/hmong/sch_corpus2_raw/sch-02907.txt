tawmtsaam, muajyeej, hab tswj.  Qhov kws puab lub qhovmuag pum ces
puab ntseeg; huas qhov kws puab lub qhovmuag tsi pum ces puab tsi
ntseeg.  Qhovntawd ces yog puab tu religion, qhov kws qha txug puab
lub zug, txujsa, hab kev nyob moog lug kws yuav tau muaj lub ntuj, lub
teb, hab lub zug kws ibtxwm muaj ntawd.
Qhov teebmeem:
1.	Cov saub nyob xyaw ntuj hab teb, tamsis puab qha tsi tau puab
tugkheej rua cov thawjcoj.  Puab paub tsi taag txug lub neej hab
txujsa rua ntawm saab ntsujplig, thaus puab coj lug siv rua ntawm saab
nqajtawv ces nwg tsi muaj kuab.  Yuav tsum tau paub has tas yog ces ua
moog, tsi yog ces thim, kuas suavdlawg coj tau hab yuav tau.  Nwg muaj
caij “khoo” (invincible) hab nwg muaj caij “muasluav” (vunerable).
2.	Cov thawjcoj tseem nyob theem siv “tsuv zug” kws tsi yog thawjcoj.
Puab tsuas tswj kuas puab tau xwb vim puab tsi muaj kev ncaajnceeg.
Puab tsi paub txug puab tugkheej.  Puab tsuas pum qhov lwm tug pum
puab xwb, ces puab yuav tsum ua kuas hum le lwm tug nyam.  Qhovntawd
yog qhov kws yuamkev ib tam dlhau moog rua ib tam.
3.	Moob pheej tsiv yog moob dlaig rua them “kaajsab” kws tsi paub
txhawj le.  Thaus lub neej hab txujsa nyuab ces ncutxug lub qub neej
(Hmong Kingdom) vim tsi paub yog rua lub neej hab txujsa.  Xaav npaum
twg los tsi pum tom hauvntej le vim cov thawjcoj coj tsi ncaaj, huas
cov saub los coj lub neej vwm quasntsuav le hab.  Thaus ntawd ces Moob
txhajle nyob rua lub neej npausuav kuas muaj tug cawmsheej (tsuv zug +
saub).  Qhov tseeb tag, Moob tsi paub Moob tugkheej.

Yog kawg! vim hmoob tsis paub hmoob tus kheej es thiaj tsis muaj tus
pab hmoob thiab.
